# SINK DINK Media Factory Plan

Goal:

Turn Paperclip into a media factory that produces upload-ready content packs.

Final output target:

- Reel video files
- Carousel image files
- Captions
- Hashtags
- Subtitle files
- Preview page or preview document
- QA report
- Upload checklist

High-level flow:

1. User gives content command.
2. CEO creates execution plan.
3. Research and trend work is prepared.
4. Scripts and carousel text are written.
5. Media file engine prepares assets.
6. QA checks output quality.
7. Final artifact pack is created.
8. User manually uploads final files.

Paperclip should provide the workflow engine, agent coordination, task tracking, review process, and artifact storage.
