# Build Roadmap

## Purpose

Track the remaining build work for the SINK DINK AI Media Factory.

## Phase A: Stability

1. Confirm deployment is healthy.
2. Confirm CEO test run works.
3. Confirm task updates work.
4. Confirm artifact output path works.

## Phase B: Organisation setup

1. Use SINK DINK as production project.
2. Use Onboarding as testing project.
3. Use bulk scripts for task setup.
4. Keep core plans in this repository.

## Phase C: Media pack MVP

1. Create first upload pack template.
2. Create carousel slide template.
3. Create basic image export plan.
4. Create reel script template.
5. Create subtitle export plan.
6. Create final pack folder structure.

## Phase D: Agent and skill setup

1. Research role.
2. Content role.
3. Media render role.
4. QA role.
5. Packager role.
6. Review role.

## Phase E: Output workflow

1. Generate content pack.
2. Generate carousel files.
3. Generate short video files.
4. Generate captions.
5. Generate hashtags.
6. Generate upload checklist.
7. Package final output.

## Phase F: Daily factory

1. Daily topic scan.
2. Daily content ideas.
3. Daily upload-ready pack.
4. Daily QA report.
5. Daily review note.

## User role

The user should mainly review the output and manually upload final files.
